Chat Conquest

A pygame based pixel art game about capturing towers and castles. The twist? Chat Conquest is hooked up to a TikTok Live Chat api which enables it to be played using Tiktok's live chat feature. The viewers are the players and they play using the livechat. This is my very first game and the assets are made entirely by me.

Vuzix Blade Object Recognizer

The Vuzix Blade is a pair of alternate reality glasses capable of displaying images directly onto the lens. While working under my lecturer, I have created a machine learning programme specifically for the Vuzix Blade Glasses using Tensorflow Lite's api to give the glasses some object recognition capabilities
